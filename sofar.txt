## How to Run Locally

1. Install Ruby and Bundler if not already installed
2. In the repository root, run:
   ```
   bundle install
   bundle exec jekyll serve
   ```
3. Visit http://localhost:4000/ChainOfNoThought/ in your browser

## Notes for Future Developers

- The CSS uses SCSS with Jekyll's built-in Sass processor
- JavaScript is kept minimal and vanilla (no frameworks) for simplicity
- All layouts use Liquid templating language
- The site is designed to be deployed on GitHub Pages
- The color scheme and typography are defined in _sass/_variables.scss
